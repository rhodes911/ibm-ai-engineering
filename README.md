## ğŸš€ Getting Started

### Prerequisites
- Python 3.10+
- Jupyter Notebook
- Git

### Setup

```bash
# Clone the repository
git clone https://github.com/rhodes911/ibm-ai-engineering.git
cd ibm-ai-engineering

# Create virtual environment
python -m venv .venv

# Activate virtual environment
# On Windows:
.\.venv\Scripts\activate
# On macOS/Linux:
source .venv/bin/activate

# Install dependencies
pip install -r requirements.txt

# Launch Jupyter
jupyter notebook
```
