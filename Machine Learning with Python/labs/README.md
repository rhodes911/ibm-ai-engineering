# Labs Directory

This directory contains hands-on Jupyter notebook labs for the IBM AI Engineering Professional Certificate course.

## Directory Structure

```
labs/
├── module-1/          # Introduction to Machine Learning labs
├── module-2/          # Linear and Logistic Regression labs
├── module-3/          # Building Supervised Learning Models labs
├── module-4/          # Building Unsupervised Learning Models labs
├── module-5/          # Evaluating and Validating ML Models labs
└── module-6/          # Final Project and Exam labs
```

## Module 2: Linear and Logistic Regression

### Labs Included:

1. **01-simple-linear-regression-lab.ipynb**
   - Implementing simple linear regression with scikit-learn
   - Training and testing on CO2 emissions dataset
   - Model evaluation with MSE, MAE, RMSE, and R²
   - Practice with different features (engine size vs fuel consumption)

2. **02-multiple-linear-regression-lab.ipynb**
   - Implementing multiple linear regression with 2+ features
   - Feature selection and correlation analysis
   - Handling multicollinearity
   - Feature standardization with StandardScaler
   - Train/test split methodology
   - 3D visualization of regression plane
   - Comparing multiple regression vs simple regression models
   - Practice exercises with engine size and fuel consumption

## Usage

Each lab notebook includes:
- Clear objectives and learning goals
- Step-by-step instructions
- Code examples with explanations
- Practice exercises with solutions
- Real-world datasets

## Running the Labs

1. Ensure you have Jupyter Notebook or JupyterLab installed
2. Install required dependencies (listed in each notebook)
3. Navigate to the appropriate module folder
4. Open the desired notebook
5. Follow the instructions and complete the exercises

## Dependencies

Common libraries used across labs:
- NumPy
- Pandas
- Matplotlib
- Scikit-learn
- Seaborn (for some modules)

Specific version requirements are listed in each notebook.
